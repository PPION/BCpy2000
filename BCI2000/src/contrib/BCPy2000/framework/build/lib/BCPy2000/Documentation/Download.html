Download

The BCPy2000 source distribution is part of the BCI2000 source distribution (under src/contrib/BCPy2000) and the newest version can be obtained using subversion. This currently requires you to register for a username, which is free and automatic: see the BCI2000 download page for details.

Alternatively you can download one of the following builds directly (but note that they may be slightly less up-to-date than the svn versions). See the changelog for a version history of the builds.

Note that binaries (foundation executables and Python package installers) are currently only available for Windows.

Option A: The Full Monty

This is the all-in-one runnable distribution. It includes:

  • a demo distribution of the BCPy2000 foundation (looking like a slimline version of a BCI2000 directory tree) without any source-code;
  • next to that, a portable standalone installation of Python itself, with all the required dependencies locked and loaded, including the BCPy2000 framework package.

The Full Monty may be the option you want if you have never installed Python beforeâ€”but be warned that it is quite large, expanding to 200 MB or more. There are various advantages and disadvantages to using the Fully Monty, about which you can read more here.

Instructions:

  • Download BCPy2000-FullMonty254-20110710.zip
  • Unzip it inside a directory that has a short path name (like C:\).
  • Go to the BCI2000/batch subdirectory.
  • Launch one of the demo batch files.
  • If you have any trouble, the first thing you should do is to read the "troubleshooting" section of the Full Monty page.
  • Maybe read the documentation so that you understand what you're seeing. Note that the second demo will not work until you've recorded at least one file with the first.

Option B: Pick and Choose

  • BCPy2000-demo-20110710.zip contains the complete BCPy2000 source+demo distribution. Like the demo in option A, it contains the three foundation executables pre-built for Windows, and the Operator.exe from the BCI2000 core.
  • In order to be able to run the demo above, you need to install Python itself and various free third-party packages. If you want, you can manage this all yourself (by consulting this page) and then install the BCPy2000 framework package from source. Or you can download all the separate binary installers for Python 2.5.4, including the one for the BCPy2000 framework, in
    python25-installers-BCPy2000-20110710.zip. You can launch INSTALL_ALL.bat, which does exactly what it says on the filename, or if you have an existing Python installation, you can pick and choose which packages you need to add.
